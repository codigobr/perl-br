O que é o Perl?

O Perl é uma linguagem de programação criada por Larry Wall em 1987.

Perl significa Pratical Extraction and Report Language.

A linguagem é usada para:

  • Desenvolvimento Web (lado do servidor),
  • Criar scripts para automatizar tarefas do sistema,
  • Desenvolvimento de software.

O que o Perl pode fazer?

  • Perl pode ser usado para criar aplicações web, rodando do lado do servidor.
  • Perl pode ser usado para automatizar tarefas do sistema.
  • Perl pode se conectar à banco de dados.
  • Perl pode ler, criar, alterar e excluir arquivos.
  • Perl pode ser usado para executar cálculos complexos matemáticos.
  • Perl pode ser usado para criar aplicativos de uma só linha ou programas mais complexos.

Por que Perl?

  • Perl funciona em diversas plataformas (Windows, Mac, Linux, etc).
  • Perl é fácil de aprender por ter uma sintaxe intuitiva.
  • Perl é interpretado, isso significa que o seu código pode ser testado rapidamente, facilitando a prototipação.
  • Perl pode ser usado para programação orientada a objetos, procedural ou funcional.

Importante saber

  • A versão mais recente do Perl é 5.32, essa versão será usada nesse tutorial.
  • O Perl 6 se tornou outra linguagem de programação, o Raku.
